💡 Use the Admin Panel to add new cards, update content, and manage your site — no code required. All changes save instantly to your browser and persist across sessions.
📋 Quick Reference — How to Add a New Resource
  1. Create your new HTML file (e.g. war-template.html) and upload it to GitHub
  2. Click ➕ Add New Card in the sidebar
  3. Fill in the card details — title, description, icon, color, URL, tags
  4. Set Status to Live and click Save Card
  5. The card appears instantly on your home page
